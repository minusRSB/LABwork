# LABwork
Лабораторная работа студента Кушхова Амира, 2025г
---
## Цель:

Цель проекта — получить понимание о структуре данных, выявить основные взаимосвязи между признаками и сделать выводы, которые могли бы быть полезны в дальнейшем анализе или принятии решений.

## Что включено:

- Предобработка данных: обработка пропусков, типы данных, проверка качества
- Исследовательский анализ (EDA): анализ распределений, визуализации, проверка гипотез
- Выводы: основные наблюдения и возможные направления дальнейшей работы

Формат: Jupyter Notebook (.ipynb).

## Используемые инструменты:

- Python
- pandas
- matplotlib / seaborn
- tensorflow
- Jupyter Notebook

## Основные шаги:

- Загрузка и первичный осмотр данных
  Посмотрел на форму, типы данных, основные статистики.

- Очистка и подготовка
  Обработал пропуски, привёл данные к корректному формату.

- Разведочный анализ
  Исследовал распределения, построил графики и таблицы, чтобы увидеть взаимосвязи.

- Формулирование выводов
  Сформулировал, что можно сказать о данных на основе анализа.

Как использовать

Открой *.ipynb в Jupyter Notebook, чтобы увидеть весь анализ и код. Советую выполнять это все либо в Jupyter, лбио в Google Colab
